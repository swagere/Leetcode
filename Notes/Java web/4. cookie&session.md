### 1. Cookie

#### 1.1 基本使用

创建：

```java
//创建cookie对象
Cookie cookie = new Cookie();
//将cookie加入响应头
resp.add(cookie);
```



服务器获取cookie：

```java
Cookie[] cookies = req.getCookies();
```



#### 1.2 cookie生命控制

最大生存时间：

setMaxAge

- 正：存入硬盘，指定秒数后过期
- 负：存入内存，浏览器关闭时cookie被删除（默认为-1
- 零：浏览器马上删除cookie



#### 1.3 cookie有效路径

用path属性，可以指定哪些路径发送哪些cookie（实现过滤效果
